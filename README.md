# Food Ordering Website

A full-stack Food Ordering Website built using React.js and Django.  
This application allows users to browse food items, place orders, and reserve tables through a clean and responsive interface.


## Technologies Used

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Django
- Django REST Framework

### Database
- MySQL


## Features

- User Signup & Login Authentication
- Responsive Food Menu UI
- Restaurant Table Reservation
- Order Placement System
- Backend API Integration
- Clean Navigation Bar
- Full-stack React + Django Architecture
